This is our list of basic HTML tags:

  1. a tag for link
  2. b tag to make bold text
  3. strong tag for bold text with emphasys
  4. body tag main HTML part
  5. br tag for break
  6. div tag it is a division or part of an HTML document
  7. i tag to make an italic text
  8. img tag for images in document
  9. ol tag is an ordered list, ul tag for an unordered list
  10. li tag is a list item in bulleted (ordered list)
  11. p tag for paragraph
  12. span tag to style part of text
